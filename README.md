## Run
```bash
    flask --app ./src/app.py run --debug #--host=0.0.0.0
```

## Create and activate venv (windows)
```bash
    py -3 -m venv .api_flask
    source ./.api_flask/Scripts/activate
``` 

## Install dependencies
```bash 
    pip install Flask
    pip install pandas
```
